Python and Frameorks for python


Python (он же пито́н, он же па́йтон, он же удав, он же зачам мне плюсы когда есть он ) — высокоуровневый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью, ориентированный на повышение производительности разработчика, читаемости кода и его качества, а также на обеспечение переносимости написанных на нём программ. Язык является полностью объектно-ориентированным в том плане, что всё является объектами. Необычной особенностью языка является выделение блоков кода пробельными отступами. Недостатками языка являются зачастую более низкая скорость работы и более высокое потребление памяти написанных на нём программ по сравнению с аналогичным кодом, написанным на компилируемых языках, таких как C или C++. Python является мультипарадигмальным языком программирования, поддерживающим императивное, процедурное, структурное, объектно-ориентированное программирование, метапрограммирование и функциональное программирование.

Лучшей утилитой для работы с питоном можно назвать продукцию на основе IntelliJ IDEA JetBrains PyCharm(Это кроссплатформенная интегрированная среда разработки для языка программирования Python, разработанная компанией JetBrains на основе IntelliJ IDEA. Предоставляет пользователю комплекс средств для написания кода и визуальный отладчик) в ней автоматически устанлено множество фреймворков для различных задач

У питона есть фножество фрэмйм ворков вот одни из самых популярных и лучших:

Django

Django — один из самых топовых веб-фреймворков на языке Python. Он подходит для разработчиков разного уровня: от новичков до профессионалов. Django поможет вам сэкономить время на написание кода и сделать его более эффективным. Плюсы: Маршрутизация URL. Миграция схем баз данных. Работа с базами данных: PostgreSQL, MySQL, SQLite и Oracle. Поддержка веб-серверов. Использование объектно-реляционного роутера (ORM). Родной движок шаблонов.

CherryPy

CherryPy — это микрофреймворк. CherryPy — это объектно-ориентированный HTTP-микрофреймворк для Python. С его помощью вы можете быстро разрабатывать веб-приложения. Он является компонентом лучшего фреймворка TurboGears для Python. Плюсы: Позволяет запускать несколько HTTP-серверов одновременно. Наличие гибкой системы плагинов. Готовые инструменты для аутентификации, кэширования, кодирования, сессий, статического контента. Работает на Android. HTTP/1.1 совместимый веб-сервер с пулом потоков WSGI.

Flask

Это еще один микрофреймворк. Flask имеет модульную конструкцию. Его также можно использовать при разработке приложений. Он удобен и позволяет выбирать расширения. Плюсы: Наличие встроенного отладчика, собственного сервера и множества шаблонов. Может подключаться к любой ORM. Flask имеет собственный движок Jinja2, но этот фреймворк может работать и с другими. Основан на Unicode. Совместим с WSGI 1.0. Совместим с Google App Engine.

TurboGears

Это удобный Python с открытым исходным кодом, самый популярный веб-фреймворк, используемый разработчиками для создания веб-приложений, работающих с данными и базами данных. Он подходит как для новичков, так и для профессионалов. Фреймворк считается отличной альтернативой Django. Плюсы: Элементы командной строки. Полнофункциональная структура. Поддержка Genshi, Repoze, SQLAlchemy, SQLObject и WebOb. Поддержка валидации с помощью FormEncode. Интеграция библиотеки MochiKit JavaScript. Использование ToscaWidgets (это облегчает координацию дизайна фронтенда и развертывания сервера). Поддержка гибкой и мощной ORM, простых и понятных шаблонов PasteScript. Поддержка большого количества баз данных. Есть возможность расширения фреймворка путем создания плагинов или добавления к существующим.

Tornado

Tornado классифицируется как асинхронный наиболее популярный веб-фреймворк python, который, благодаря неблокирующему обмену данными, способен одновременно поддерживать множество пользовательских соединений. Он идеально подходит для задач, требующих длительного соединения с каждым пользователем. Плюсы: Имеет собственный механизм аутентификации, при необходимости можно подключить сторонние. Поддержка переводов и локализации. Работа в режиме реального времени.
